# Rolling Dice

A clean mini web project with a 3D animated dice.
Click **Roll Dice** to spin the cube and get a random value from 1 to 6.

## Preview

Built with pure HTML, CSS, and JavaScript - no frameworks, no build setup.

## Run

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Press **Roll Dice**.

### Optional (local server)

If Node.js is installed:

```bash
npx serve .
```

Then open the URL shown in the terminal (usually `http://localhost:3000`).

## Files

- `index.html` - markup
- `style.css` - styling and 3D animation
- `index.js` - dice roll logic
